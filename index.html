Predicting Student Anxiety using ML Models

I built an interactive Streamlit web app to predict student anxiety levels using K-Means clustering and classification models (XGBoost, Random Forest). The project applies PCA for dimensionality reduction, data preprocessing, and achieved high accuracy on multi-class predictions. Users can input features such as Age, Study Hours, and GAD/SWL/SPIN scores to get real-time predictions of Low, Moderate, or High anxiety.
